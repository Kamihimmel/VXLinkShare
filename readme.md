# VXLinkShare

VXLinkShare is a browser extension for Chrome and Firefox that cleans shared links by stripping tracking parameters and automatically converting links for Discord, Slack, Telegram, and favorite link formats.

## What it does

- Removes extra tracking and analytics parameters from shared URLs.
- Converts links into sharing-friendly formats used by Discord, Slack, Telegram, and other messaging or bookmarking services.
- Helps preserve privacy and keep shared links clean and readable.

## Supported platforms

- Chrome extension
- Firefox extension
- Future support planned for:
  - iOS
  - Android

## Key features

- Automatic link cleanup when copying or sharing links.
- Smart rewriting for common chat and social platforms.
- Focused on improving link quality without changing destination behavior.

## Why use VXLinkShare

Sharing links often includes unwanted tracking tags, campaign data, or long query strings. VXLinkShare removes these extras and helps share only the URL that matters.
